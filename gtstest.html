anyCommerce + Google Trusted Stores test page

Basic idea how this should work:

In Rich Internet Application (RIA) HTML page loads only once. And HTML DOM initializes only once.
After that any user interaction happens without the page reload - we just make ajax calls to send/fetch the data and delete/insert html nodes into the existing DOM - using javascript.
So, Google Trusted Stores badge code in RIA also initializes only once... But, unfortunately, that's not enough!

After you press "Create Test Order" button, we'll show "Thank You" note to the user. There will be no page reload - javascript will insert "gts-o" data sections
